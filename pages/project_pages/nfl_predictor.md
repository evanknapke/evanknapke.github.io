# NFL Predictor
---

[Check out the source code](https://github.com/evanknapke/NFL-Predictor)

## Description

Pulls live NFL stats from the web to predict outcomes of the upcoming schedule.

## Features
- Predicts:
    - Spread margin
    - Score of both teams
    - Total score for over/under bets
- View results in a terminal
- Optionally save results to a text file
- Easily adjustable weight for accuracy
- Easily adjustable decimals for user preference
