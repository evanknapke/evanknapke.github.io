# Server Number Guesser
---

[Check out the source code](https://github.com/evanknapke/Server-Number-Guesser)

## Description

Creates a server that is able to handle multiple clients. Creates a dialogue with the client(s) and guesses a number of their choosing.

Program sets up a server and handles multiple clients. When a client is connected to the server it begins dialogue and eventually asks the client to think of a number 1-1000. The program loops until the number is guessed, and then the server is closed.
