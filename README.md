# evanknapke.github.io

My portfolio webpage

## Development
#### To serve to localhost:
run `bundle exec jekyll serve` in project directory

- if you receive this error: `/jekyll/commands/serve/servlet.rb:3:in 'require': cannot load such file -- webrick (LoadError)`
  - You are likely using the wrong version of ruby
  - run `rvm use 2.7 --default` to switch the correct version

open `http://localhost:4000` in a web browser to view
